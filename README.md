# LogicLab

LogicLab es una herramienta web para simular compuertas lógicas, generar tablas de verdad y simplificar expresiones booleanas mediante álgebra booleana, mapas de Karnaugh y método tabular.

## Características

- Simulación visual de compuertas lógicas
- Compuertas AND, OR, NOT, NAND, NOR, XOR y XNOR
- Compuertas configurables con varias entradas
- Tabla de verdad automática
- Simplificación por mapa de Karnaugh
- Simplificación por álgebra booleana paso a paso
- Método tabular Quine–McCluskey
- Conexión visual mediante cables
- Deshacer y rehacer cambios con Ctrl + Z / Ctrl + Y
- Exportación del circuito como SVG

## Uso

Abre la página publicada en GitHub Pages y comienza a construir circuitos directamente desde el navegador.
